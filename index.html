Ce qu’est le Web & ses principes de fonctionnement

Le World Wide Web (WWW), ou simplement le Web, est un système d'information où les documents et autres ressources web sont identifiés par des URL, liés entre eux par des hyperliens et accessibles via Internet.

Fonctionnement (Client-Serveur)

Le principe fondamental repose sur l'architecture Client-Serveur :

  • Le Client (Navigateur) envoie une requête HTTP (demande une page).
  • Le Serveur traite la requête et renvoie les fichiers correspondants (HTML, CSS, JS, images).
